# Power BI Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data using Power BI and provides insights into sales performance, profitability, and regional trends.

## Features

- Data Cleaning with Power Query
- Star Schema Data Modeling
- DAX Measures
- Interactive Dashboard
- Sales Trend Analysis
- Regional Performance Analysis
- Category Analysis
